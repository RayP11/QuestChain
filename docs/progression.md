# Progression

Your agent isn't a tool you reset. It's a companion you build.

Every agent starts at **Level 1** and grows stronger through real work — the more it does, the more capable and experienced it becomes. Like a Pokémon, it evolves over time. Unlike a Pokémon, it's actually doing your tasks while you sleep.

---

## Leveling up

Every conversation, every web search, every file it edits, every quest it completes — it all counts. Your agent earns XP automatically just by being used. Hit enough XP and it levels up, all the way from **Level 1 to Level 20**.

The more specialized the work, the more it grows. A Builder that ships code levels up faster than one that just chats. An Explorer that digs deep across multiple sources earns more than one that skims.

---

## Achievements

Along the way, your agent earns achievements — permanent milestones that reflect what it's actually done.

`First Strike` · `Awakening` · `Seasoned` · `Veteran` · `Legend` · `Century` · `Old Timer` · `Iron Will` · `Bibliophile` · `Web Walker` · `Globe Trotter` · `Blacksmith` · `Demolition` · `Archivist` · `Grand Planner` · `Polymath` · `Speed Demon` · `Centurion` · `Busy Bee` · `Road Runner`

Each one tells a story. A **Bibliophile** has read hundreds of files. A **Globe Trotter** has browsed the web extensively. A **Blacksmith** has built real things.

You can see exactly what your agent has been up to — what it reaches for most, how much ground it's covered, and what it's earned.

## Build your party

Create multiple agents, each with their own class, name, and progression. A high-level Explorer that's done thousands of searches. A Builder that's shipped real features. A Keeper that knows your entire notes archive inside and out.

The more you use them, the more they become *yours*.
